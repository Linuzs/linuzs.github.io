2019-06-19|平台搭建
字数总计: 1.2k|阅读时长: 4 分钟

前言

前一段时间学校社团组织了一次CTF小比赛，当时就是用的CTFd这个平台。无奈的是最近CTFd更新了，所以旧版本的包我就安装不上了。最近研究了一下新版的CTFd(v2.1.2)，顺便还把新版的CTFd汉化了。总体来说，现在的CTFd功能越来越强大了，变成了团队赛以及个人赛两种模式，在管理员后台也可以给一些用户进行奖励加分的操作等等…

环境要求

  • Docker
  • docker-compose
  • nginx
  • python2.7
  • python-pip
